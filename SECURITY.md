# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x | ? |
| <1.0.0 | ? |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, report vulnerabilities privately through GitHub Security Advisories or contact the repository owner.

## Disclosure Process

1. Report received
2. Validation
3. Fix developed
4. Security release
5. Public disclosure
